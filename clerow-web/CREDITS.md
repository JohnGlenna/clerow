# Third-party asset credits

## Game-art icons

Clerow's gamification surfaces (streaks, XP, quests, achievements, levels,
pricing tiers) use icons from **[game-icons.net](https://game-icons.net)**,
consumed via the [`react-icons`](https://react-icons.github.io/react-icons/)
package under its `gi` (Game Icons) set.

These icons are licensed under
**[Creative Commons Attribution 3.0 (CC BY 3.0)](https://creativecommons.org/licenses/by/3.0/)**.
Free for commercial use; attribution is required.

Attribution is surfaced to end users in the site footer
(`components/landing/Landing.tsx`) and recorded here. Individual icons are
authored by various contributors to game-icons.net — see the per-icon author
credits at <https://game-icons.net/about.html#authors>.

The curated set Clerow uses is defined in `components/GameIcon.tsx`. UI chrome
(navigation, buttons, form controls) uses the separate hand-built line-icon set
in `components/Icon.tsx`, which is original to this project.
